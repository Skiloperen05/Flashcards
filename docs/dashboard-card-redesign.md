# Dashboard card redesign

Forslag for nye fagkort på Hjem:

- Kortene bør vise fagkode, navn og en kort beskrivelse av hva faget dekker.
- Hvert kort bør vise verktøy-chips som A-besvarelser, eksamensanalyse, forelesningsnotater, innleveringsoppgaver, memoar og kompendium.
- Ring/prosent kan gjøres mindre slik at innholdet får mer plass.
- CTA bør være tydelig: Start øving, Se fagside eller Se plan.

Dette dokumentet er lagt på egen branch for videre iterasjon.
